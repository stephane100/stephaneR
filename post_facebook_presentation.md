# Posts Facebook — Présentation freelance dans groupe

## ⚙️ Règles d'or pour Facebook
- **1ère ligne = hook** (sinon les gens scrollent)
- **Court** : les longs posts perdent 70% des lecteurs
- **Émojis pour scanner** mais pas trop (3-5 max)
- **CTA clair** : dis exactement ce que tu veux ("MP", "commentez", etc.)
- **Pas de lien dans le post lui-même** : Facebook réduit la portée des posts avec liens externes → mets le lien **en commentaire** ou en MP
- **Poste en heure de pointe** : 12h-14h ou 19h-22h (heure Mada)

## 💡 Conseils stratégiques pour le post

### Avant de publier
1. **Lis les règles du groupe** (certains interdisent l'autopromo, ou demandent un format spécifique)
2. **Regarde les posts qui marchent** dans le groupe → reproduis le format
3. **Mets une photo de toi** (la photo de profil pro double l'engagement)
4. **Tague-toi** sur ton portfolio si possible

### Au moment du post
1. **Commente ton propre post immédiatement** avec le lien portfolio (Facebook pénalise les posts avec liens externes dans le corps)
2. **Réponds à TOUS les commentaires** dans les 30 min qui suivent — ça booste la portée
3. **Demande à 2-3 amis** de liker dans les 10 min (effet boule de neige sur l'algo)

### Après le post
1. **Reviens dans 4-6h** pour répondre aux commentaires/MP
2. **Ne supprime pas** même s'il y a peu d'engagement — ça reste indexé pour les recherches
3. **Re-poste une variante** dans 2-3 semaines si tu veux relancer (jamais le même texte)

### Erreurs à éviter
- ❌ Trop d'émojis (3-5 max)
- ❌ TOUT EN MAJUSCULES (perçu comme du spam)
- ❌ Tarifs dans le post (à discuter en MP, ça force le contact)
- ❌ "Je suis le meilleur" sans preuve (mets toujours du chiffré : "2 apps", "7 ans", "3 plateformes")
- ❌ Lien externe dans le corps du post (mets-le en commentaire)
- ❌ Trop long (>15 lignes = perte de 50% des lecteurs)

**Ma recommandation : Version 1.**

C'est le meilleur équilibre entre crédibilité (chiffres, preuves) et engagement (CTA clair). Le hook "2 apps iOS publiées sur l'App Store" arrête le scroll instantanément dans un groupe freelance — la majorité n'en ont jamais publié une seule.

## 🔄 Variantes pour les semaines suivantes

Une fois ton 1er post fait, tu peux varier les angles :

### Post #2 — Cas concret
> "J'ai livré la semaine dernière une plateforme e-commerce pour [client]. Voici ce qui m'a fait gagner 2 semaines sur le projet : [astuce technique]. Si vous avez un projet similaire, MP."

### Post #3 — Question-piège (engagement booster)
> "Question pour les entrepreneurs du groupe : c'est quoi votre plus gros frein quand vous voulez créer une plateforme web ou une app ? (Le budget ? Trouver un dev fiable ? La maintenance ?) Je suis curieux de vos réponses."

### Post #4 — Avant/Après visuel
> Capture d'écran de Looqs ou Evolya + "De l'idée au store en X mois. Si vous avez une idée d'app et besoin d'un dev pour la concrétiser, MP."

### Post #5 — Témoignage client
> Si un client te laisse un bon retour, repost-le avec son accord.

L'idée : **1 post par semaine max, varier les angles** (compétence / cas concret / question / résultat / témoignage).
